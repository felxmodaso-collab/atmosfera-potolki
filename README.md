# АТМОСФЕРА — натяжные потолки

Многостраничный коммерческий сайт услугового бизнеса. Полный функционал многосекционного лендинга в стиле центров отделки: квиз-калькулятор, портфолио объектов, прайс-сетка, FAQ, формы заявок, мессенджеры, политика обработки ПДн по 152-ФЗ.

**Live:** https://felxmodaso-collab.github.io/atmosfera-potolki/

## Стек

- **Next.js 14** (App Router, Static Export)
- **TypeScript**
- **Tailwind CSS** (custom design tokens, серифная Cormorant Garamond + sans Inter)
- **Vanilla CSS animations** (никаких runtime libs — лёгкий бандл)
- **Pollinations.ai (FLUX)** — все интерьерные фото сгенерированы через бесплатный API

## Страницы

| Маршрут | Что внутри |
|---------|------------|
| `/` | Hero, mini-калькулятор, типы потолков, портфолио-тизер, процесс, преимущества, отзывы, FAQ, CTA |
| `/services` | 8 типов потолков с детальным описанием, фото, ценой, фичами |
| `/calculator` | Полноразмерный калькулятор + 6-шаговый квиз со скидкой |
| `/portfolio` | 12 кейсов с модалкой деталей (общий план + крупный план + бюджет + клиент) |
| `/prices` | 3 пакета, прайс по типам, прайс по опциям |
| `/about` | Команда, шоурум, реквизиты юр.лица, сертификаты |
| `/contacts` | Форма обратной связи, мессенджеры, FAQ-аккордеон, карта Yandex |
| `/privacy` | Полная политика обработки ПДн по 152-ФЗ (12 разделов) |

## Конверсионные элементы

- **Квиз 6 шагов** с прогресс-баром и фиксацией скидки 10%
- **Калькулятор** с live-обновлением сметы (тип помещения → площадь → периметр → тип потолка → опции)
- **Контактная модалка** в трёх вариантах (заявка / звонок / замерщик), глобальный provider, доступна с любой страницы
- **Плавающая кнопка контактов** в правом нижнем углу: WhatsApp / Telegram / MAX / звонок
- **Sticky header** с CTA «Замер бесплатно»
- **Формы** с валидацией, обязательной галочкой согласия на обработку ПДн (по 152-ФЗ от 01.09.2025)
- **CTA-баннеры** между секциями, четыре варианта формулировок

## 152-ФЗ комплаенс

- Политика обработки ПДн на отдельной странице
- Маркированная отдельная галочка согласия в каждой форме
- Ссылка на политику в каждой форме
- Все необходимые реквизиты юр.лица в политике
- Упоминание уведомления Роскомнадзора в течение 24 ч при инциденте

## Дизайн-система

- Цвета: тёплый off-white фон (#FAF8F4), кремовые секции, графитовый текст, тёплый бронзовый акцент (#B8946A), глубокий ink (#0E0F11)
- Типографика: серифная Cormorant Garamond для заголовков, sans Inter для UI
- Скруглённые pill-кнопки, мягкие тени, hover-эффекты
- Mobile-first сетка, бургер-меню на <lg

## Локальный запуск

```bash
npm install
npm run dev
# http://localhost:3000
```

## Билд и деплой

```bash
npm run build      # → ./out (static export)
```

Деплой автоматический: push в `main` → GitHub Actions → GitHub Pages.

## Регенерация изображений

Если нужно перегенерировать интерьерные фото:

```bash
python generate_images_pollinations.py
```

Pollinations.ai (FLUX) — бесплатно, без квот. На 37 фото уходит 15-25 минут.
